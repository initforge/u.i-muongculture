## Tính năng

- Trang nội dung văn hóa với React Router
- Lưu trữ media bằng Vercel Blob
- Thiết kế responsive

## Cài đặt

```bash
git clone https://github.com/initforge/u.i-muongculture.git
cd u.i-muongculture
npm install && npm run dev
```
